# Problem Type Classification - Advent of Code 2016

This document classifies Advent of Code 2016 problems by their primary problem type.

## Classification Categories

- **Data Processing**: Problems involving parsing, transforming, filtering, or aggregating data
- **Grid (2D Matrix)**: Problems involving 2D grid/matrix manipulation, pathfinding, or spatial operations
- **Games & Algorithms**: Problems involving game simulation, game rules, or algorithmic challenges like recursion, backtracking, dynamic programming
- **Others**: Problems that don't fit into the above categories

## Detailed Analysis

### Day1 - Taxicab
**Type**: Data Processing

**Description**: 
- Follows navigation instructions (turn left/right, move steps)
- Calculates Manhattan distance from origin
- Part 2 finds first location visited twice

**Key Operations**:
- Direction tracking and rotation
- Position tracking
- Set-based visited tracking (Part 2)

**Note**: Uses 2D coordinates but not a grid structure - just position tracking

### Day2 - BathroomSecurity
**Type**: Grid (2D Matrix)

**Description**:
- Navigates keypad grid following movement instructions
- Part 1: 3x3 numeric keypad
- Part 2: 5x5 diamond-shaped keypad with invalid positions

**Key Operations**:
- Grid-based navigation
- Boundary checking
- Position validation

### Day3 - Triangles
**Type**: Data Processing

**Description**:
- Validates triangles based on side lengths
- Part 1: Validates each line as triangle
- Part 2: Groups by columns (vertical triangles)

**Key Operations**:

- Triangle inequality validation
- Data grouping

### Day4 - Security
