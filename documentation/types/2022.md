# Problem Type Classification - Advent of Code 2022

This document classifies Advent of Code 2022 problems by their primary problem type.

## Classification Categories

- **Data Processing**: Problems involving parsing, transforming, filtering, or aggregating data
- **Grid (2D Matrix)**: Problems involving 2D grid/matrix manipulation, pathfinding, or spatial operations
- **Games & Algorithms**: Problems involving game simulation, game rules, or algorithmic challenges like recursion, backtracking, dynamic programming
- **Others**: Problems that don't fit into the above categories

## Detailed Analysis

### Day1 - CalorieCounting
**Type**: Data Processing

**Description**: 
- Groups calorie values by empty lines
- Sums calories per group
- Part 1: Finds maximum, Part 2: Sums top 3

**Key Operations**:
- Grouping by separator
- Summing values
- Sorting and limiting

### Day2 - RockPaperScissors
**Type**: Data Processing

**Description**:
- Calculates Rock-Paper-Scissors game scores
- Part 1: Interprets second column as shape
- Part 2: Interprets second column as outcome

**Key Operations**:
- Strategy parsing
- Score calculation
- Outcome determination

### Day3 - RucksackReorganization
**Type**: Data Processing

**Description**:
- Finds common items in rucksack compartments
- Part 1: Common item in two halves
- Part 2: Common item in groups of 3

**Key Operations**:
- String splitting
- Set intersection
- Character priority calculation

### Day4 - CampCleanup
**Type**: Data Processing

**Description**:
- Checks range overlaps between pairs
- Part 1: Checks if one range fully contains another
- Part 2: Checks if ranges overlap at all

**Key Operations**:
- Range parsing
- Subset checking (Part 1)
- Overlap detection (Part 2)

**Note**: Creates full range lists which could be optimized

### Day5 - SupplyStacks
**Type**: Data Processing

**Description**:
- Moves crates between stacks
- Part 1: Moves one at a time (reverses order)
- Part 2: Moves multiple at once (preserves order)

**Key Operations**:
- Stack initialization
- Stack operations (push/pop)
- Command parsing

### Day6 - TuningTrouble
**Type**: Data Processing

**Description**:

- Part 2: 14 unique characters

**Key Operations**:
