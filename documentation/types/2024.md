# Problem Type Classification - Advent of Code 2024

This document classifies Advent of Code 2024 problems by their primary problem type.

## Classification Categories

- **Data Processing**: Problems involving parsing, transforming, filtering, or aggregating data
- **Grid (2D Matrix)**: Problems involving 2D grid/matrix manipulation, pathfinding, or spatial operations
- **Games & Algorithms**: Problems involving game simulation, game rules, or algorithmic challenges like recursion, backtracking, dynamic programming
- **Others**: Problems that don't fit into the above categories

## Detailed Analysis

### Day1 - HistorianHysteria
**Type**: Data Processing

**Description**: 
- Processes two lists of numbers, calculates distances between sorted pairs
- Part 2 counts occurrences and multiplies

**Key Operations**:
- Parsing and splitting input
- Sorting lists
- Calculating distances and aggregating
- Counting occurrences

### Day2 - SafeReports
**Type**: Data Processing

**Description**:
- Validates sequences of numbers
- Checks if differences are in valid range and if sequence is sorted
- Part 2 tries removing one element to make sequence valid

**Key Operations**:
- Sequence validation
- Difference checking
- Sorting validation
- Subsequence generation

### Day3 - MullItOver
**Type**: Data Processing

**Description**:
- Parses instructions from text using regex
- Filters and processes mul() instructions
- Part 2 uses accumulator pattern with enable/disable flags

**Key Operations**:
- Regex pattern matching
- Instruction parsing
- Filtering and aggregation
- Stateful accumulation

### Day4 - SoupLetter
**Type**: Grid (2D Matrix)

**Description**:
- Searches for word "XMAS" in 2D grid in 8 directions
- Part 2 searches for "MAS" in X pattern (diagonal intersections)

**Key Operations**:
- Grid traversal
- Directional search (8 directions)
- Recursive word matching
- Pattern detection

### Day5 - PrintQueue
